# 機能一覧（基本設計）

## 1. 認証機能

| 機能ID  | 機能名       | 概要                             | MVP |
| ------- | ------------ | -------------------------------- | --- |
| AUTH-01 | ユーザー登録 | メールアドレスとパスワードで登録 | ○   |
| AUTH-02 | ログイン     | 認証処理                         | ○   |
| AUTH-03 | ログアウト   | セッション破棄                   | ○   |
| AUTH-04 | プロフィール編集 | 名前・メール・学習目標を更新  | ○   |
| AUTH-05 | パスワード変更 | パスワードを更新               | ○   |
| AUTH-06 | 退会         | アカウントを削除                 | ○   |

---

## 2. 試験管理機能

| 機能ID | 機能名   | CRUD | 概要                   | MVP |
| ------ | -------- | ---- | ---------------------- | --- |
| EX-01  | 試験登録 | C    | 試験を登録する         | ○   |
| EX-02  | 試験一覧 | R    | 登録済み試験を一覧表示 | ○   |
| EX-03  | 試験編集 | U    | 試験情報更新           | ○   |
| EX-04  | 試験削除 | D    | 試験削除               | ○   |
| EX-05  | 基準点設定 | U   | 合格基準点と目標得点を管理 | ○ |

## 5. 分析機能

| 機能ID | 機能名             | 概要                 | MVP |
| ------ | ------------------ | -------------------- | --- |
| AN-01  | 月別学習時間グラフ | 月単位で合計時間表示 | ○   |
| AN-02  | 直近7日グラフ      | 直近7日の学習時間表示 | ○  |
| AN-03  | 継続指標           | 学習継続状況を表示   | ○   |
| AN-04  | 科目別学習時間     | 科目ごとの学習量を表示 | ○ |
| AN-05  | 模試点数推移グラフ | 模試結果の推移表示   | ○   |
| AN-06  | 最新模試レーダー   | 科目別得点を表示     | ○   |
| AN-07  | 基準点比較表示     | 合格基準点と目標得点との差を表示 | ○ |
